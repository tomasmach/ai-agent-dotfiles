# Third-party software

This repository does not vendor the external skills and plugins listed in `manifests/`. Their names, upstream URLs, paths, and recorded revisions are provided for reproducibility only. Each project remains governed by its upstream license.

Claude Code, Codex, CLIProxyAPI, and their plugins are separate projects and are not covered by this repository's MIT license.

If an upstream source disappears, the manifest still documents what was installed, but this repository intentionally does not act as an unlicensed mirror.

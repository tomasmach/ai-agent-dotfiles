# Icon Styles Reference

Available styles for icon generation. These are fetched live from the API, but this reference helps the skill describe options to users when the API is unavailable.

## Common Styles

- **Flat** — Clean, solid-color icons with no gradients or shadows. Modern and minimalist.
- **Gradient** — Icons with smooth color transitions. Vibrant and eye-catching.
- **Minimalist** — Stripped-down designs with essential elements only. Elegant and simple.
- **3D** — Icons with depth, lighting, and perspective effects. Bold and dynamic.
- **Glossy** — Shiny, reflective surfaces reminiscent of classic iOS icons.
- **Clay** — Soft, matte 3D appearance like clay or plasticine. Playful and modern.

## Tips for Good Icon Descriptions

- Be specific about the main visual element: "a rocket" not "something fast"
- Mention the mood: "playful", "professional", "bold", "elegant"
- Reference colors if the user's app has brand colors
- Keep it concise: 1-2 sentences max
- Avoid text in icons — small text is unreadable at app icon sizes
